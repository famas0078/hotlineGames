Перед запуском проекта не забудьте скачать и установить NODE JS по ссылке https://nodejs.org/en
    1) После установки выполняем команду -> npm i 
    2) Далее мы проверили наличие всех библиотек командой выше, теперь пишем -> npm run dev
    3) Проект запущен на порте  5174 либо 5173, все пишется в консоли, ссылка для повторения http://localhost:5174/
